# Financial Market Prediction & Anomaly Detection Platform

## Overview
Enterprise-grade finance analytics platform for market prediction and anomaly detection. This portfolio-ready application processes financial time-series data, generates predictions using multiple ML models, detects anomalies, and provides AI-driven insights.

## Features

### Core Features
- **CSV Data Upload**: Robust CSV handling with automatic column type detection
- **Data Cleaning**: Automatic missing value handling, outlier detection, and standardization
- **Feature Engineering**: 20+ technical indicators including moving averages, RSI, MACD, Bollinger Bands
- **Machine Learning Models**:
  - Linear Regression for price prediction
  - K-Nearest Neighbors for price movement classification
  - Neural Network for return prediction
  - Gradient Boosted Trees for volatility prediction
- **Anomaly Detection**: Isolation Forest algorithm for detecting unusual patterns
- **AI-Generated Insights**: Dynamic plain English insights based on data analysis
- **Interactive Dashboard**: Professional Streamlit interface with dark blue theme

### Professional Dashboard
- Clean, enterprise-grade design (no emojis)
- Dark blue (#0B3D91), white (#FFFFFF), gold accent (#FFD700) color scheme
- Interactive charts with Plotly
- Multiple tabs for different analysis views
- Export functionality for all generated data
